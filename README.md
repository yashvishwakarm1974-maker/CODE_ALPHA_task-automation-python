# 📁 JPG File Automation Script

This Python script automatically moves all `.jpg` image files from a source folder to a destination folder.

## Features

- Automatically detects `.jpg` files
- Creates destination folder if it doesn't exist
- Moves all image files
- Displays total files moved
- Beginner-friendly

## Technologies

- Python 3
- os
- shutil

## How to Run

```bash
python move_jpg_files.py
```

Enter:

- Source folder path
- Destination folder path

The program will automatically move all `.jpg` files.

## Concepts Used

- File Handling
- os Module
- shutil Module
- Loops
- If-Else
